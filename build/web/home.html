Information
  1. Select to upload a valid GeoJSON file to display on the base map.

  2. Select to hide the boundaries of the regions displayed on the map.

  3. Select to display the boundaries of the regions displayed on the map.

  4. Select to display edit mode of the map.

  5. Note that in edit mode, when is selected and edit mode is reverted back to default mode, all changes will be lost so do select to download the GeoJSON file. This file can then be uploaded via the button available in default mode to be displayed on the map."

  6. Select to logout of the application.
